# TeLlevaApp

TeLlevaApp es una aplicación móvil desarrollada como **proyecto académico**, cuyo objetivo es facilitar el **transporte compartido entre estudiantes de Duoc UC**, permitiendo coordinar viajes de manera segura, colaborativa y económica, especialmente para quienes asisten a clases en horarios vespertinos.

---

## Descripción

La aplicación permite que los estudiantes puedan **ofrecer o buscar viajes compartidos** con compañeros que tengan rutas similares, promoviendo la seguridad, el ahorro en transporte y la colaboración dentro de la comunidad estudiantil.

El enfoque principal del proyecto estuvo en el desarrollo de las funcionalidades esenciales y en la aplicación de buenas prácticas de desarrollo vistas durante la carrera.

---

## Funcionalidades implementadas

- Registro e inicio de sesión de usuarios.
- Publicación de viajes disponibles por parte de conductores.
- Visualización y búsqueda de viajes publicados.
- Solicitud de cupos para viajes.
- Gestión básica de viajes y usuarios.

Estas funcionalidades fueron **desarrolladas, probadas y se encontraban operativas** al momento de la evaluación académica.

## Instalación y ejecución

### Requisitos previos

Antes de ejecutar el proyecto, asegúrate de tener instalado:

- Node.js (versión LTS recomendada)
- npm
- Ionic CLI
- Android Studio (para Android) o Xcode (para iOS)

Instalar Ionic CLI:

```bash
npm install -g @ionic/cli
```
Instalar dependencias:
```
npm install
```
Ejecutar en entorno de desarrollo:
```
ionic serve
```
Ejecutar en Android:
```
ionic capacitor add android
ionic capacitor sync
ionic capacitor run android
```
Nota: Es necesario tener Android Studio configurado correctamente.
